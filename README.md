# JobHub 

## Project Description

JobHub is a web-based platform where employers can post job and internship opportunities, and users can search and apply for them.
The system supports two types of users: employers and job seekers.

## Technologies

* Angular (Frontend)
* Django REST Framework (Backend)
* HTML, CSS, JavaScript

---

## Features

### Authentication

* User registration and login
* Role-based access (Employer / Candidate)

### Job Management

* Create job postings (Employer)
* Edit and delete jobs
* View job listings

### Job Search

* Search jobs by title and category
* Filter by salary and type (job / internship)

###  Applications

* Apply for jobs
* View application status
* Employers can see applicants

---


## 👥 User Roles

### Employer

* Post jobs
* Manage job listings
* View applicants

### Job Seeker

* Browse jobs
* Apply to jobs
* Track applications

---

## Purpose

The goal of this project is to create a simple and modern job search platform that connects employers with job seekers efficiently.
